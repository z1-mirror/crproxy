Note
         Only for docker.io
         
         Prefixes the image used with 'myzero1.xyz'
         myzero1/mybin ---> myzero1.xyz/myzero1/mybin

         Or 
         
         Setting /etc/docker/daemon.json 
         "registry-mirrors": [
            "https://myzero1.xyz"
         ]
